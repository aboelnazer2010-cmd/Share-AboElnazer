# Share AboElnazer — P2P File Sharing & Chat

A decentralized peer-to-peer file sharing and chat application built with Next.js, WebRTC, and Socket.IO. 

### Features:
- **Obsidian Terminal Design:** Modern, sleek interface.
- **Privacy First:** P2P connection, no servers store your data.
- **No Signup:** Instant rooms and random usernames.
- **Multilingual:** Supports Arabic and English.

### Local Development:
```bash
npm install
npm run dev
```

### Deployment:
Deployed on Hugging Face Spaces using Docker.
